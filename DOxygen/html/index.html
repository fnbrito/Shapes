Program Introduction

The Shapes application is meant to show how objects inherit from other classes. As well, this program is the first to use the DOxygen documenting tool.

Project Requirements

The shapes project must be separated into one superclass called Shape and two subclasses called Circle and Square. The subclasses will inherit the datamembers and the public functions. Additionally they'll have three added virtual functions. Encapsulation must be respected.
